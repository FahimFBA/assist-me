## Installation

### Prerequisites

- NodeJS
- NPM
- Firebase CLI
- Tailwind CSS
- Git
- VS Code
- Chrome

### Steps

1. Clone the repository

```bash
git clone https://github.com/FahimFBA/assist-me.git
```

2. Install dependencies

```bash
cd assist-me
npm install
```

3. Create a Firebase Project and add your own environment variables in `.env` file.
4. Run the app

```bash
npm start
```
